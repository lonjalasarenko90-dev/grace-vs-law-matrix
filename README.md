# Grace vs Law Matrix Test

Інтерактивний тест, який показує, у якій духовній «матриці» мислення живе людина:

- Система Закону
- Система Благодаті
- Змішана Матриця

Проєкт створений як демонстраційний прототип для відео, дослідження мислення та богословських ідей про закон і благодать.

---

# Як працює тест

Тест містить **15 запитань** зі шкалою від 1 до 5.

1–2 → відповіді мислення **закону**  
3 → нейтрально  
4–5 → відповіді мислення **благодаті**

Алгоритм рахує два показники:

- Law %
- Grace %

І визначає результат:

### Система Закону
Law ≥ 65%  
Grace ≤ 45%

### Система Благодаті
Grace ≥ 65%  
Law ≤ 45%

### Змішана Матриця
усі інші випадки
